Web Visualizations

In this project I used HTML and CSS to create a Web dashboard of data visulizations that measured cities' weather.

The powerful grid tool, Bootstrap, was used to drive sizing and positioning of elements.

This allows for web display resizing to happen driven by the user to optimize display.

Providing a Web dashboard of analysis, such as this, allows findings to be more easily accessed and shared.
